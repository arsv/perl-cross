Cross-compiling perl

Intro Download Usage Design Modules Hints Testing

Current version of perl-cross is 1.5.1
Most recent supported perl version: perl-5.38.1
Most recent supported cperl version: cperl-5.26.1

https://github.com/arsv/perl-cross/raw/releases/perl-cross-1.5.1.tar.gz

To use, unpack over an appropriate perl distribution, overwriting the original Makefile.
Use one of supported perl versions; there are version-specific patches inside.

For older releases, check GitHub releases branch.
